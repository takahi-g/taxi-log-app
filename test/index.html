📖 TAXI Log Pro 使い方ガイド

📱 1. スマホのホーム画面に追加

アプリのようにワンタップで起動できるようにしましょう！

  • iPhone (Safari): 下の「共有ボタン(□に↑)」➡「ホーム画面に追加」をタップ。※追加する際に「Webアプリとして開く」というチェック項目がある場合は、チェックが入っていることを確認してください。
  • Android (Chrome): 右上のメニュー「︙」➡「ホーム画面に追加」または「アプリをインストール」をタップ。

📅 2. 月はじめの設定 (初期設定)

毎月1回、以下の設定をするだけで今月の目標売上が自動セットされます。

  1. 「⚙️ 設定」タブで設定対象年月を選択。
  2. 出勤カレンダーで、今月の出勤シフトの日付をタップして黄色（出勤予定日）にします。
  3. 「平日標準目標」と「金土祝前目標」を入力。
  4. 自動的に月全体の「目標売上」と「出勤日数」が計算されて保存されます。

💰 3. 毎日の売上追加

「💰 売上計算」タブでの毎日の操作です。

  • 降車時の税込売上（例: 4800, 12000など）をテンキーで入力して「売上を追加する」を押すだけ。
  • 今日の目標（設定した平日目標または金土目標）に自動で連動し、「今日の残りノルマ」がリアルタイムに減っていきます。

☕ 4. 休憩時間の計測・編集

乗務中の休憩もしっかり記録・管理できます。

  • 休憩に入るときに「休憩に入る」を押し、終わったら「休憩終了」を押すだけで自動で分数がカウントされます。
  • 「何時〜何時まで休憩したか」の履歴は、履歴リストの「✏️（編集）」ボタンから後からいつでも開始・終了時刻を修正できます。

📊 5. 履歴と手取り給与の確認

「📝 詳細履歴」タブで今月の全データが集計されます。

  • カレンダーで日別の売上の有無を確認できます。
  • 月間売上から会社の歩合給テーブルに沿った「推定の歩合給」を自動計算します。
  • さらに売上時給だけでなく、歩合率を考慮した「実際にもらえる手取り時給」も売上計算タブに大きく表示され、モチベーション維持に役立ちます。

💾 6. データのバックアップとExcel出力

万が一のデータ保存や、表計算ソフトで編集したい場合に使用します。

  • データをバックアップ: タップすると暗号化された引き継ぎコードがコピーされます。メモ帳などに貼り付けて保存してください。
  • データを復元: コピーしておいたコードを貼り付けるだけで、一瞬でこれまでのデータが元の状態に戻ります。
  • Excel用CSV書き出し: これまでの全売上履歴を、ExcelやGoogleスプレッドシートで文字化けせず直接綺麗に開ける表データ（CSVファイル）としてスマホやPCへ保存します。
